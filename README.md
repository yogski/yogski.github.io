## Yogi Saputro Personal Profile Web
I have tried many stuff. Template engines, components, React, etc.
Good old vanilla JS is reliable enough for this profile.

### Pages
- `index.html`: main page.
- `projects.html`: list of projects.
- `systems.html`: list of clients I have been working with.
- `articles.html`: list of articles I wrote in another plaftorm.
- `/stock-opname`: Simple, local-only stock opname web app. Surprisingly useful for small stores. Seamless integration with barcode scanner. [Check it out](https://yogski.github.io/stock-opname/).
- `/knights-journey`: A simple game to demonstrate usage of my npm package `headless-game-grid`. It's tutorial + live demo. [Check it out](https://yogski.github.io/knights-journey/).
